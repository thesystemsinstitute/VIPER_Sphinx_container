exhale - Working Example
=========================

This page demonstrates the capabilities of **exhale**, a Sphinx extension that automatically generates API documentation hierarchies for C++ projects using Breathe and Doxygen.

.. note::

   **About exhale**
   
   exhale creates a class and file hierarchy for C++ projects, automating the generation of Breathe directives and creating a browsable API structure. It works on top of Breathe and Doxygen.
   
   - **Package**: exhale
   - **Purpose**: Automatic C++ API hierarchy generation
   - **Use Case**: Large C++ codebases needing organized documentation
   - **Tutorial**: :doc:`../tutorials/packages/exhale`


Installation
------------

Basic Installation
~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   pip install exhale

The extension is already installed in this environment:

.. code-block:: python

   import exhale
   print(f"Exhale version: {exhale.__version__}")

Dependencies
~~~~~~~~~~~~

Exhale requires both Breathe and Doxygen:

.. code-block:: bash

   # Install all dependencies
   pip install sphinx breathe exhale
   
   # Install Doxygen (system package)
   sudo apt-get install doxygen  # Ubuntu/Debian
   brew install doxygen           # macOS

Configuration
-------------

Basic Configuration
~~~~~~~~~~~~~~~~~~~

Add to your ``conf.py``:

.. code-block:: python

   # Enable required extensions
   extensions = [
       'breathe',
       'exhale'
   ]
   
   # Breathe configuration
   breathe_projects = {
       "MyProject": "./doxyxml/"
   }
   breathe_default_project = "MyProject"
   
   # Exhale configuration
   exhale_args = {
       # Required arguments
       "containmentFolder": "./api",
       "rootFileName": "library_root.rst",
       "rootFileTitle": "Library API",
       "doxygenStripFromPath": "..",
       
       # Doxygen configuration
       "createTreeView": True,
       "exhaleExecutesDoxygen": True,
       "exhaleDoxygenStdin": "INPUT = ../include"
   }

Advanced Configuration
~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: python

   # conf.py - Advanced exhale settings
   exhale_args = {
       # Output configuration
       "containmentFolder": "./api",
       "rootFileName": "library_root.rst",
       "rootFileTitle": "Complete API Reference",
       
       # Doxygen integration
       "exhaleExecutesDoxygen": True,
       "exhaleDoxygenStdin": """
           INPUT            = ../include ../src
           EXCLUDE_PATTERNS = */internal/* */test/*
           EXCLUDE_SYMBOLS  = *_internal *_impl
       """,
       
       # Tree view settings
       "createTreeView": True,
       "exhaleUseDoxyfile": False,
       
       # Listing configuration
       "fullToctreeMaxDepth": 3,
       "listingExclude": [r'.*_p\.h$'],  # Exclude private headers
       
       # Page generation
       "unabridgedOrphanKinds": {"class", "struct"},
       
       # Path handling
       "doxygenStripFromPath": "..",
       
       # Custom CSS
       "customSpecificationsMapping": {
           "class": "custom-class",
           "function": "custom-function"
       }
   }

Using Existing Doxyfile
~~~~~~~~~~~~~~~~~~~~~~~

If you have an existing Doxyfile:

.. code-block:: python

   # conf.py
   exhale_args = {
       "containmentFolder": "./api",
       "rootFileName": "library_root.rst",
       "rootFileTitle": "API",
       
       # Use existing Doxyfile
       "exhaleExecutesDoxygen": True,
       "exhaleUseDoxyfile": True,
       "exhaleDoxygenStdin": "INPUT = ../include"  # Override INPUT only
   }

Practical Examples
------------------

Simple Library
~~~~~~~~~~~~~~

**Project Structure**:

.. code-block:: cpp

   // include/mylib/math.h
   namespace mylib {
       /**
        * @brief Mathematical constants and functions
        */
       class Math {
       public:
           /// Calculate square root
           static double sqrt(double x);
           
           /// Pi constant
           static constexpr double PI = 3.14159265359;
       };
   }

**Exhale Configuration**:

.. code-block:: python

   # conf.py
   exhale_args = {
       "containmentFolder": "./api",
       "rootFileName": "api_root.rst",
       "rootFileTitle": "MyLib API",
       "doxygenStripFromPath": "..",
       "exhaleExecutesDoxygen": True,
       "exhaleDoxygenStdin": "INPUT = ../include/mylib"
   }

Template Library
~~~~~~~~~~~~~~~~

**C++ Templates**:

.. code-block:: cpp

   // include/containers.h
   namespace containers {
       /**
        * @brief Generic vector container
        * @tparam T Element type
        */
       template<typename T>
       class Vector {
       public:
           void push_back(const T& value);
           T& operator[](size_t index);
           size_t size() const;
       };
       
       /**
        * @brief Generic map container
        * @tparam K Key type
        * @tparam V Value type
        */
       template<typename K, typename V>
       class Map {
       public:
           void insert(const K& key, const V& value);
           V& operator[](const K& key);
       };
   }

**Configuration**:

.. code-block:: python

   exhale_args = {
       "containmentFolder": "./api",
       "rootFileName": "containers_api.rst",
       "rootFileTitle": "Container Library",
       "createTreeView": True,
       "exhaleExecutesDoxygen": True,
       "exhaleDoxygenStdin": """
           INPUT = ../include/containers.h
           EXTRACT_ALL = YES
       """
   }

Build Workflow
--------------

Basic Build
~~~~~~~~~~~

.. code-block:: bash

   # Exhale runs Doxygen automatically
   cd docs
   make clean
   make html

With Makefile
~~~~~~~~~~~~~

.. code-block:: makefile

   # Makefile
   SPHINX_BUILD = sphinx-build
   SOURCE_DIR = .
   BUILD_DIR = _build
   
   .PHONY: help clean html
   
   help:
   	@echo "make html  - Build HTML documentation"
   	@echo "make clean - Clean build directory"
   
   html:
   	$(SPHINX_BUILD) -b html $(SOURCE_DIR) $(BUILD_DIR)/html
   	@echo "Build finished. Open $(BUILD_DIR)/html/index.html"
   
   clean:
   	rm -rf $(BUILD_DIR) api/ doxyxml/

CMake Integration
~~~~~~~~~~~~~~~~~

.. code-block:: cmake

   # CMakeLists.txt
   find_package(Sphinx REQUIRED)
   
   add_custom_target(docs
       COMMAND ${SPHINX_EXECUTABLE} -b html
               ${CMAKE_SOURCE_DIR}/docs
               ${CMAKE_BINARY_DIR}/docs/html
       WORKING_DIRECTORY ${CMAKE_SOURCE_DIR}
       COMMENT "Building documentation with Sphinx"
   )

Advanced Features
-----------------

Tree View Generation
~~~~~~~~~~~~~~~~~~~~

Enable interactive tree view:

.. code-block:: python

   exhale_args = {
       # ... other settings ...
       "createTreeView": True,
       "minifyTreeView": True
   }

This creates collapsible hierarchy views.

Orphan Handling
~~~~~~~~~~~~~~~

Control standalone pages:

.. code-block:: python

   exhale_args = {
       # ... other settings ...
       # These kinds get their own pages even without references
       "unabridgedOrphanKinds": {
           "class", "struct", "enum"
       }
   }

Path Stripping
~~~~~~~~~~~~~~

Clean up displayed paths:

.. code-block:: python

   exhale_args = {
       # ... other settings ...
       "doxygenStripFromPath": "../..",
       # /home/user/project/include/file.h -> include/file.h
   }

Namespace Organization
~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: python

   exhale_args = {
       # ... other settings ...
       "fullApiSubSectionTitle": "API by Namespace",
       "createTreeView": True
   }

Troubleshooting
---------------

Exhale Not Running
~~~~~~~~~~~~~~~~~~

Check extension loading:

.. code-block:: python

   # conf.py - Ensure correct order
   extensions = [
       'breathe',  # Must come before exhale
       'exhale'
   ]

Doxygen Errors
~~~~~~~~~~~~~~

Debug Doxygen execution:

.. code-block:: python

   exhale_args = {
       # ... other settings ...
       "verboseBuild": True
   }

.. code-block:: bash

   # Run Sphinx with verbose output
   sphinx-build -v -b html . _build/html

Missing Classes
~~~~~~~~~~~~~~~

Ensure Doxygen processes files:

.. code-block:: python

   exhale_args = {
       "exhaleDoxygenStdin": """
           INPUT = ../include
           RECURSIVE = YES
           FILE_PATTERNS = *.h *.hpp
           EXTRACT_ALL = YES
       """
   }

GitHub Actions
~~~~~~~~~~~~~~

.. code-block:: yaml

   # .github/workflows/docs.yml
   name: Documentation
   
   on: [push, pull_request]
   
   jobs:
     build-docs:
       runs-on: ubuntu-latest
       steps:
         - uses: actions/checkout@v3
         
         - name: Install dependencies
           run: |
             sudo apt-get install -y doxygen
             pip install sphinx breathe exhale
         
         - name: Build documentation
           run: |
             cd docs
             make html
         
         - name: Deploy to GitHub Pages
           if: github.ref == 'refs/heads/main'
           uses: peaceiris/actions-gh-pages@v3
           with:
             github_token: ${{ secrets.GITHUB_TOKEN }}
             publish_dir: ./docs/_build/html
