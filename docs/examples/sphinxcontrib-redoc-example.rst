Installation
------------

Using pip
~~~~~~~~~

Install the extension:

.. code-block:: bash

   pip install sphinxcontrib-redoc

Or add to your ``requirements.txt``:

.. code-block:: text

   sphinxcontrib-redoc
   sphinx>=5.0.0
