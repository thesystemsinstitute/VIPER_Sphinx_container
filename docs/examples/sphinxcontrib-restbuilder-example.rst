Installation
------------

Using pip
~~~~~~~~~

Install the extension:

.. code-block:: bash

   pip install sphinxcontrib-restbuilder

Or add to your ``requirements.txt``:

.. code-block:: text

   sphinxcontrib-restbuilder
   sphinx>=5.0.0

-------------

~~~~~~~~~~~~~~~~~~~

Add to your ``conf.py``:

.. code-block:: python
