Installation
------------

Using pip
~~~~~~~~~

Install the extension:

.. code-block:: bash

   pip install sphinxcontrib-websupport

Or add to your ``requirements.txt``:

.. code-block:: text

   sphinxcontrib-websupport
   sphinx>=5.0.0
