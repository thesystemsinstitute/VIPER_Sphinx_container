Getting Package Versions
-------------------------

To see the exact versions installed in your container, run:

.. code-block:: bash

   docker run kensai-sphinx pip list | grep -i sphinx

Or for all packages:

.. code-block:: bash

   docker run kensai-sphinx pip list

.. note::
   The versions listed as "Latest" will be the most recent versions available 
   at the time of building the container. To see exact versions, use the commands above.
