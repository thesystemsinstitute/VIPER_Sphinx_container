What is N2G?
-----------
N2G provides features for:

- Core functionality
- Integration with Sphinx
- Configuration options
- Advanced features
- Best practices

The N2G package provides functionality for working with Network to Graph converter.


Installation
------------

Install via pip:

.. code-block:: bash

   pip install N2G

Configuration
-------------

Add to your Sphinx ``conf.py``:

.. code-block:: python

   extensions = [
       'N2G',
   ]
