This tutorial demonstrates how to use pydata-sphinx-theme, a PyData community theme.

What is PyData Sphinx Theme?
---------------------------

PyData Sphinx Theme provides features for:

- Core functionality
- Integration with Sphinx
- Configuration options
- Advanced features
- Best practices

Installation
------------

Install via pip:

.. code-block:: bash

   pip install pydata-sphinx-theme

Configuration
-------------

Add to your Sphinx ``conf.py``:

.. code-block:: python

   extensions = [
       'pydata_sphinx_theme',
   ]
