What is Python Markdown Graphviz?
--------------------------------
Python Markdown Graphviz provides features for:

- Core functionality
- Integration with Sphinx
- Configuration options
- Advanced features
- Best practices

The python-markdown-graphviz package provides functionality for working with Graphviz extension for Markdown.


Installation
------------

Install via pip:

.. code-block:: bash

   pip install python-markdown-graphviz

Configuration
-------------

Add to your Sphinx ``conf.py``:

.. code-block:: python

   extensions = [
       'python_markdown_graphviz',
   ]
