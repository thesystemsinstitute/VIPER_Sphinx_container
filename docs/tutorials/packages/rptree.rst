What is RPTree?
--------------

RPTree provides features for:

- Core functionality
- Integration with Sphinx
- Configuration options
- Advanced features
- Best practices


The rptree package provides functionality for working with directory tree generator.

Installation
------------

Install via pip:

.. code-block:: bash

   pip install rptree

Configuration
-------------

Add to your Sphinx ``conf.py``:

.. code-block:: python

   extensions = [
       'rptree',
   ]
