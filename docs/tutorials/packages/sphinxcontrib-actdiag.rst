What is Sphinxcontrib-ActDiag?
-----------------------------

Sphinxcontrib-ActDiag provides features for:

- Core functionality
- Integration with Sphinx
- Configuration options
- Advanced features
- Best practices

Installation
------------

Install via pip:

.. code-block:: bash

   pip install sphinxcontrib-actdiag

Configuration
-------------

Add to your Sphinx ``conf.py``:

.. code-block:: python

   extensions = [
       'sphinxcontrib_actdiag',
   ]
