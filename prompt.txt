Generate HTML documentation for this container.
The title page:
	- explains the purpose of this container
	- link to the Sphinx documentation
	- Explanation how to use the installed tools in this container from your project
	- Links to the following sub pages:
	- "Sphinx packages" with table listing all installed Sphinx-related Python packages with the following columns
            - "Name" for the names of the package
            - "Version" for the version of the Python package installed
            - "Pypi" for the links to the Pypi page for the Python package
            - "docs" for the links to the generated pdoc3 doumentation for the Python package
            - "manual" link to the online documentation  for the Python package or tool
            - "descr" for a one line description of the Package
			- "Tutorial" a link to the included tutorial for this package.
	- Tutorial page on how to use all Sphinx features, with a link to an example page. Embed youtube videos where possible.
	- Tutorial page for every Sphinx package installed , with a link to an example page. Embed youtube videos where possible.

Embed the generated documentation in the container.
Run the generated HTML pages on a small embedded  webserver based Python http.server.
Expose the port to allow expternal users to show the web pages in their browser
